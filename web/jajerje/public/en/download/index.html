Installation

The installation is very simple. If you follow these steps, you won’t get lost. I promise.

  1. Download the file from the button above. You will see a file with the zip extension being downloaded. We’re on the right track!
  2. Extract the file to your favorite location. To do this, right-click on the file and extract it with your preferred tool.
  3. Open the folder that appeared after extracting the file.
  4. Run FeedTheLeader.exe.
  5. And enjoy!
